# ItID-Scrapping-JN
Pequeño ejemplo de web Scrapping para la clase de Introducción a la Ingeniería de Datos.
En este pequeño ejemplo se extrajo el contenido de las noticas de la página www.lajornadamaya.mx/, en el apartado de noticias en maya
"k'iintsil", el rango de las noticias que se abarcó fueron todos los meses del año 2019.
Luego dicho contenido se guardó en un CSV, con las columnas: Título, Fecha, Contenido.

(Disclaimer: el contenido no está del todo limpio)
